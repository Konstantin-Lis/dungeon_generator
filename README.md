# dungeon_generator
Генератор подземелий на любой вкус

Три способа генерации подземелий:
1. Клеточными автоматами
2. С помощью Шума Перлина
3. Туннельный алгоритм (его я пока не представляю, надо узнать подробнее)

## Генерация при помощи Шума Перлина
1. Потенциально хорошо поддерживает небольшие перепады высот
   (если где-то есть небольшая возвышенность, на которую можно забраться или ямка)
2. Либо создаёт большие пространства, либо множество разделённых помещений,
   в этом плане карта особо требует ручной доводки
3. Изначально заточена на создание наземной поверхности (с холмами и впадинами)

## Генерация клеточными автоматами
Надо попробовать создавать несколько карт и накладывать их друг на друга.
Так потенциально можно смоделировать перепады высоты

Или попробовать трёхмерную генерацию, где показатель высоты очень мал (1-2-3).
Тогда можно взглядом сверху определять высоты


1. Изначально имеет структуру двумерного поля, где каждая клетка это 0 или 1,
    в силу этого хорошо подходит для создания плоских лабиринтов
2. Я попробую что-то придумать для создания перепадов местности
    или многоуровневых лабиринтов, но для перепадов лучше Шум Перлина.
    Для многоуровневых - не знаю, надо изучать тему

## Туннельный алгоритм
1. Потенциально хорошо заточен именно под прямоугольные помещения по типу комнат и переходов в здании
2. Возможно, получится использовать для многоуровневых помещений, надо изучать тему
